# Soluções Beecrowd (URI Online Judge)

Este repositório contém minhas soluções pessoais para os problemas da plataforma [Beecrowd](https://www.beecrowd.com.br/), um site para praticar programação competitiva e resolver desafios de algoritmos.

O objetivo principal é documentar meu progresso, registrar as diferentes lógicas aplicadas e servir como um portfólio dos meus estudos em algoritmos e estruturas de dados.

## Linguagens Utilizadas

As soluções neste repositório são (ou serão) implementadas principalmente na linguagem Java

Eventualmente, outras linguagens podem ser adicionadas.

## Estrutura de Pastas

Para manter a organização, o repositório segue a estrutura de categorias do próprio Beecrowd. Cada problema resolvido está dentro da sua respectiva categoria e possui uma pasta própria.

### Categorias Principais

* Iniciante
* Ad-Hoc
* Estruturas e Bibliotecas
* Strings
* Matemática
* Paradigmas
* Grafos
